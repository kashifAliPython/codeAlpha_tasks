# Video Game Sales Analysis Project

## Project Overview
Is project me Video Game Sales dataset ko analyze kiya gaya hai using Python. Is ka main purpose data ko clean karna, us ka analysis karna aur different visualizations ke through insights nikalna hai.

---

## Dataset Details
Is dataset me video games se related information di gayi hai jaise:
- Game Name
- Platform
- Year
- Genre
- Publisher
- Regional Sales (NA, EU, JP, Other)
- Global Sales

---

## Project Goals
- Dataset ko clean karna
- Data ka basic analysis karna
- Important insights nikalna
- Graphs aur charts ke zariye data ko samajhna

---

## Task 1: Data Cleaning
Is step me:
- Missing values ko handle kiya gaya
- Duplicate records remove kiye gaye
- Dataset ko analysis ke liye ready kiya gaya

---

## Task 2: Data Analysis (EDA)
Is step me:
- Top selling games check kiye gaye
- Popular genres identify kiye gaye
- Gaming platforms ka analysis kiya gaya
- Total aur average sales calculate ki gayi

---

## Task 3: Data Visualization
Is project me ye visualizations banaye gaye:
- Top 10 Best Selling Games (Bar Chart)
- Most Popular Genres (Bar Chart)
- Top Gaming Platforms (Bar Chart)
- Global Sales Distribution (Histogram)
- Genre wise Sales Share (Pie Chart)

---

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

---

## Key Findings
- Action aur Sports games sab se zyada popular hain
- Kuch hi platforms market ko dominate karte hain
- Sales ka zyada hissa limited top games se aata hai
